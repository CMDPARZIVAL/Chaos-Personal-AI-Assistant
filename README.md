# 🤖 CHAOS AI Assistant

[![Python Version](https://img.shields.io/badge/python-3.7%2B-blue.svg)](https://www.python.org/downloads/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A powerful, voice-activated AI assistant built with Python that combines the intelligence of **Google Gemini AI**, seamless **Spotify integration**, and robust **system control capabilities**.

---

## ✨ Features

* **🗣️ Voice Recognition:** Speak naturally to control your computer.
* **🧠 AI Conversations:** Powered by Google Gemini 1.5 Flash for smart, contextual responses.
* **🎵 Music Control:** Play your favorite songs directly from Spotify.
* **⚙️ System Control:** Manage applications, volume, and core system functions hands-free.
* **🌐 Web Navigation:** Open websites and perform web searches instantly.
* **💾 Persistent Memory:** Remembers conversation context for fluid interactions.
* **🔊 Text-to-Speech:** Natural, human-like voice responses.

---

## 🎯 Capabilities

### Voice Commands
* `"Play [song name]"` - Play music from Spotify.
* `"Open [website/application]"` - Launch apps or websites.
* `"Close [application]"` - Terminate running applications.
* `"Search for [query] on the browser"` - Initiate a web search.
* `"What's the time?"` - Get the current time.
* `"Increase/decrease volume"` - Control system audio levels.
* `"Shutdown/restart"` - Manage system power states.

### AI Chat
* Natural conversations with context memory.
* Task assistance, brainstorming, and general queries.
* Persistent conversation history.
